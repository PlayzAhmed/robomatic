Multiply 1 Digit

Multiplication is like fast addition. It helps us find the total when we have groups of the same number.
For Example if you have 3 groups of 4 apples, multiplication helps you quickly find that you have 12 apples in total.

Strategies To Use In Multiplication

Repeated Addition
It indicates to adding the number to itself many times
Example :
3 x 3 = 3 + 3 + 3 = 9

Equal Groups
