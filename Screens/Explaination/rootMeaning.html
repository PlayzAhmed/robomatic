What is Root?

Think of a root as a magical number. When you use this magic number in a certain way, it helps you find another number.

⦁ Square Root (√):

The square root is like finding the number that, when you multiply it by itself, gives you a special number.
- Example: √9 is like asking, "What number times itself equals 9?" The magic number here is 3 because 3 x 3 = 9.

⦁ Cube Root (∛):

The cube root is similar, but it's the magic number that, when you multiply it by itself twice, gives you a certain number.
- Example: ∛8 is like asking, "What number times itself twice equals 8?" The magic number here is2 because 2 x 2 x 2 = 8.

⦁ General (∛n):

⦁ For any positive number 'n', the 'n-th root' is like finding the magic number that, when you use it in a special way 'n' times, gives you a particular number.
- Example:∛27 is like asking,"What number times itself three times equals 27?" The magic number here is 3 because
3 x 3 x 3 = 27.

In short, a root is like a math wizard helping us find special numbers in a magical way!
